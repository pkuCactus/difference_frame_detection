# 状态定义

## 状态
- INIT: 初始化配置与资源
- CONNECTING: 建立RTSP连接
- CHECK_CAPABILITY: 检查相机能力
- CAMERA_DETECTION_MODE: 使用相机检测能力
- LOCAL_DETECTION_MODE: 使用本地检测能力
- DIFFERENCE_ANALYSIS: 差异分析
- EVENT_ANALYSIS: 事件分析
- UPDATE_REF: 更新参考帧
- RECONNECTING: 断流重连
- ERROR: 异常状态

## 状态转移
- INIT->CONNECTING
- CONNECTING 成功 -> CHECK_CAPABILITY
- CHECK_CAPABILITY 支持 -> CAMERA_DETECTION_MODE
- CHECK_CAPABILITY 不支持 -> LOCAL_DETECTION_MODE
- CAMERA_DETECTION_MODE 检测到目标 -> DIFFERENCE_ANALYSIS
- LOCAL_DETECTION_MODE 检测目标 -> DIFFERENCE_ANALYSIS
- DIFFERENCE_ANALYSIS 有差异 -> EVENT_ANALYSIS
- EVENT_ANALYSIS 完成 -> UPDATE_REF
- UPDATE_REF 完成 -> 对应检测模式
- 任意状态发生 RTSP 断流 -> RECONNECTING
